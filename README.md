# AngularSkeleton

This is a base project for an Angular app. Quickstart with almost everything you need.

- Latest Angular
- Routing
- ESLint
- Prettier
- NgRx-Store
- Docker (multi-stage build)
- nginx config
- .gitignore

Usual scripts:

- ng - ng
- start - local development server
- build - production build
- test - unit tests
- lint - linting
- e2e - end-to-end test
- precommit - precommit script for formatting
- format:fix - fix formatting in staged files
- format:project - format the whole project
- format:check - check formatting

# How to use it

Clone and use it as your own Angular project.
